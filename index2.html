برنامه نویسی چیست؟

برنامه‌نویسی فرآیند نوشتن دستورالعمل‌هایی است که کامپیوتر بتواند آن‌ها را اجرا کند. این دستورالعمل‌ها به زبان‌های برنامه‌نویسی نوشته می‌شوند، که مانند زبان‌های انسانی، قواعد و syntax خاص خود را دارند. کامپیوترها فقط زبان ماشین (صفر و یک) را می‌فهمند، اما برنامه‌نویسان از زبان‌های سطح بالا مانند Python، JavaScript یا Java استفاده می‌کنند تا کد بنویسند، و سپس کامپایلر یا اینترپرتر آن را به زبان ماشین ترجمه می‌کند.
